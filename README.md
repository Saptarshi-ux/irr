# Understanding IRR (Internal Rate of Return) and Cash Flow Signs

## What is IRR?

**IRR (Internal Rate of Return)** is the discount rate at which the **Net Present Value (NPV) of a project becomes zero.**

In simple words:

> IRR tells the expected annual return generated by a project based on its future cash flows.

---

## Mathematical Definition

The NPV formula is:

```text
NPV = CF0/(1+r)^0 + CF1/(1+r)^1 + CF2/(1+r)^2 + ... + CFn/(1+r)^n
```

For IRR:

```text
NPV = 0
```

Where:

```text
CF = Cash Flow at a particular time
r  = IRR (discount rate)
t  = Time period
```

---

# Why is Initial Investment Negative in IRR?

The initial investment is negative because it represents a **cash outflow**.

When a company starts a project:

- Money goes OUT today → Negative cash flow
- Money comes IN later → Positive cash flow

Example:

```text
Time 0:
Investment = -₹100000

Year 1:
Return = +₹25000

Year 2:
Return = +₹30000

Year 3:
Return = +₹40000
```

So the cash flow sequence becomes:

```python
[-100000, 25000, 30000, 40000]
```

The negative sign represents money invested.

---

# Mathematical Explanation

Suppose:

Initial investment:

```text
CF0 = -1000
```

Future returns:

```text
CF1 = 600
CF2 = 600
```

The IRR equation becomes:

```text
0 = -1000 + 600/(1+r) + 600/(1+r)^2
```

The negative investment balances against the future positive returns.

After solving:

```text
IRR ≈ 13.1%
```

Meaning:

The project generates around **13.1% annual return.**

---

# Why Can't Initial Investment Be Positive?

Suppose we write:

```python
[1000, 600, 600]
```

Then:

```text
0 = 1000 + 600/(1+r) + 600/(1+r)^2
```

Every term is positive.

A positive value cannot become zero:

```text
Positive + Positive + Positive ≠ 0
```

Therefore, IRR calculation requires at least:

```text
One negative cash flow
and
One positive cash flow
```

# Project Decision Using IRR

Businesses compare:

```text
IRR vs Required Rate of Return
```

## Case 1: Accept Project

Example:

```text
IRR = 18%
Required Return = 12%
```

Comparison:

```text
18% > 12%
```

Decision:

```text
ACCEPT PROJECT
```

Reason:

The project generates more return than required.

---

## Case 2: Reject Project

Example:

```text
IRR = 9%
Required Return = 12%
```

Comparison:

```text
9% < 12%
```

Decision:

```text
REJECT PROJECT
```

Reason:

The project does not generate enough return.

---

# How Companies Determine Required Return?

Required return is usually based on:

## 1. Cost of Capital

Companies use money from:

- Debt (Loans)
- Equity (Investors)

The combined cost is called:

```text
WACC (Weighted Average Cost of Capital)
```

Formula:

```text
WACC = (E/V × Re) + (D/V × Rd × (1-T))
```

Where:

```text
E  = Equity
D  = Debt
V  = Total Capital
Re = Cost of Equity
Rd = Cost of Debt
T  = Tax Rate
```

---

## 2. Risk Adjustment

Higher risk projects require higher returns.

Example:

```text
Low risk project:
Required return = 10%

High risk project:
Required return = 25%
```

---

# Final Decision Rule

```text
IRR > Required Return
        |
        ↓
    Project status -- → ACCEPT


IRR < Required Return
        |
        ↓
    Project status -- → REJECT
```

---

# Simple Summary

| Term | Meaning |
|---|---|
| IRR | Return generated by the project |
| Required Return | Minimum return expected by investors |
| Decision | Compare IRR with Required Return |

A project is accepted only when it generates a return higher than the return investors demand.
